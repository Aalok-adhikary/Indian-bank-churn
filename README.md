# 🧠 ChurnMind India - Bank Customer Churn Prediction

ChurnMind India is an AI-powered Streamlit web application designed to predict bank customer churn using an Artificial Neural Network (ANN) model.

## 🚀 Features
- Predicts customer churn probability
- Interactive and user-friendly dashboard
- ANN-based simulation model
- Indian demographic customization
- Real-time analytics and visualization

## 📂 Project Structure
```
ChurnMind_India_Streamlit/
│── app.py
│── requirements.txt
│── README.md
```

## ⚙️ Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/ChurnMind-India.git
cd ChurnMind-India
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the Application
```bash
streamlit run app.py
```

## 🌐 Streamlit Cloud Deployment
1. Upload this project to GitHub.
2. Visit https://share.streamlit.io
3. Connect your repository.
4. Select `app.py` and deploy.

## 📊 Model Details
- Model Type: Artificial Neural Network (ANN)
- Activation Functions: ReLU and Sigmoid
- Optimizer: Adam
- Accuracy: 86% (Simulated Case Study)
